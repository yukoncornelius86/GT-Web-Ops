# GT Web Ops Local Lab
Local-first stack for GT Cafe and GT Collective: static site preview, Blog Studio, n8n+Postgres, docs, and deployment templates.

## Quick start
1. Copy `.env.example` to `.env` and fill placeholders.
2. `docker compose up -d --build`
3. Open: static preview `http://localhost:8080`, Blog Studio `http://localhost:8787`, n8n `http://localhost:5678`.

See `docs/USER_IMPLEMENTATION_GUIDE.md` first.
